OpenH264-Sys
===================

.. contents:: 


前置条件
----------

轻确保你的操作系统已经有了以下命令行程序可供使用:

*   make
*   git

构建
-----------

.. code:: bash
    
    git clone --recurse-submodules -j8 https://github.com/LuoZijun/rust-openh264-sys.git
    cargo build
    cargo test

------------
